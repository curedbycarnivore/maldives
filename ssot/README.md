# WebStorm SSOT — READ ONLY

These files are the source of truth for the maldives editor configuration.
**Never modify these files.** They are direct copies from WebStorm 2025.2.

- `keymaps/leet hax.xml` — custom keymap (parent: Mac OS X 10.5+)
- `colors/Tomorrow Night Eighties.icls` — color theme (Inconsolata 18pt, bg #2d2d2d)

To refresh from live WebStorm:
```
cp ~/Library/Application\ Support/JetBrains/WebStorm2025.2/keymaps/leet\ hax.xml ssot/keymaps/
cp ~/Library/Application\ Support/JetBrains/WebStorm2025.2/colors/Tomorrow\ Night\ Eighties.icls ssot/colors/
chmod -w ssot/keymaps/leet\ hax.xml ssot/colors/Tomorrow\ Night\ Eighties.icls
```
